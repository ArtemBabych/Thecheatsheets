Syntax and Selectors

Class and ID Selector

The class selector targets elements with a specific class attribute. It is denoted by a period (.) followed by the class name. Use it when styling multiple elements that share the same styling characteristics. Multiple elements can have the same class.
The ID selector targets a single element with a specific ID attribute. It is denoted by a hash (#) followed by the ID name. Use it when styling a unique element on the page. IDs should be unique within a page.

Groups of CSS Selectors

Groups of CSS selectors allow you to apply the same styles to multiple selectors in a single rule. This is useful when you want to style several elements with similar styling characteristics.
Selector Groups Syntax: To group selectors, you separate them with commas (,). The styles within the curly braces {} will then apply to all the selectors in the group.

Chaining Selector


You can chain different selectors, for example: if your paragrpah has an class red you can chain them. Just add them to each other, like this p.red and your p selectors with only class red will change appropriately to your properties and value of this chaining selecotor.

Type Selector


In CSS, type selectors, also known as element selectors, target HTML elements based on their tag names. They are one of the simplest selectors and apply styles to all instances of a specific HTML element.

Class selector


CSS class selectors are used to style HTML elements based on their class attribute. Unlike type selectors, which target all instances of a specific HTML element, class selectors allow you to selectively apply styles to elements with a particular class.

Selector Specificity


In CSS, selector specificity determines which style rule takes precedence:
Inline styles have the highest specificity. ID selectors are more specific than class selectors, attribute selectors, and pseudo-classes. Type selectors have the lowest specificity. When conflicting styles exist, the browser applies the rule with higher specificity, and in case of a tie, the rule that appears last in the stylesheet wins.

Descendant selector


A descendant selector in CSS is used to target an element that is a descendant of another specified element, regardless of the number of levels between them. It is denoted by placing a space between the selectors.

Link Element


The HTML <link> element is used to link external resources to an HTML document. It is commonly used to link stylesheets (CSS files), but it can also be used for other purposes, such as linking to icon files, alternate stylesheets, or external scripts.

Write CSS in Separate Files

Separation of HTML and CSS is a good practice for better organization and maintainability, especially in larger projects. It allows you to keep your styles in one place and makes it easier to manage and update your website's appearance.

Write CSS in HTML File

When writing CSS code within the <style> tags in the <head> part of an HTML document, it's essential to follow the correct syntax and structure. The CSS code consists of selectors and declaration blocks. Selectors target HTML elements, and the declaration block, enclosed in curly braces {}, contains property-value pairs separated by semicolons. Comments in CSS are written using /* */ and provide explanations or notes about the code.

Inline Style


Inline styles in HTML involve applying CSS styles directly within the HTML tags using the style attribute. This approach is useful for applying specific styles to individual elements.

Visual Rules

When it comes to creating engaging and aesthetically pleasing websites, visual rules play a crucial role in defining the appearance and layout of web elements. Cascading Style Sheets (CSS) is the language used to apply these visual rules, providing web developers with the power to control everything from colors and fonts to spacing and responsiveness.

Visual rules in web design refer to the set of instructions that dictate how elements on a webpage should look. These rules encompass a wide range of styling options, allowing developers to tailor the visual experience for users.


CSS declarations

CSS declarations are the building blocks of styles. A CSS declaration consists of a property and a value, and it defines how a particular HTML element should be styled.

Property: This is the attribute of an HTML element that you want to style. Examples include color, font-size, margin, padding, etc.

Value: This is the specific setting for the property. It can be a numeric value, a color, a measurement unit, or one of several predefined keywords.

Font Size


Setting the font size is a common CSS declaration used to control the size of text within HTML elements. The font-size property determines the size of the font, and it can be specified using various units.

Background Color


Setting the background color is a common CSS declaration used to define the color of an element's background. The background-color property is employed for this purpose.

!important Rule


The !important rule in CSS is a way to give a declaration more weight, making it override other declarations even if they have higher specificity or appear later in the stylesheet. It is often used when you want to ensure that a particular style takes precedence.

Opacity

The opacity property in CSS is used to control the transparency of an element. It takes a value between 0 and 1, where 0 is completely transparent (invisible), and 1 is completely opaque (fully visible). Values between 0 and 1 represent varying levels of transparency.

Font Weight

The font-weight property in CSS is used to set the thickness or boldness of the characters in a text element. It accepts various values, allowing you to specify different levels of boldness.

Text Align


The text-align property in CSS is used to set the horizontal alignment of text content within an element. It defines how the inline content of a block container is aligned within the block container's content box.

Setting foreground text color in CSS

In CSS, the color property is used to set the foreground text color of an element. You can specify the color using various formats such as color names, hexadecimal codes, RGB values, or HSL values.

Resource URLs


Resource URLs typically refer to the web addresses (URLs) of various resources used in web development, such as images, stylesheets, scripts, and other files. These URLs are used to locate and load these resources into web pages. In CSS, the url() function is used to wrap resource URLs. These can be applied to several properties such as the background-image.

Background Image


In CSS, you can set a background image for an element using the background-image property.

Font Family


In Cascading Style Sheets (CSS), you can specify font families for text using the font-family property. The font-family property allows you to define a prioritized list of font family names and/or generic family names for an element.

The Box Model

Topic Description Example Applicable Properties
1. box-sizing property Defines how the total width and height of an element is calculated, including padding and border. Default value is content-box. box-sizing: content-box; box-sizing
2. box-sizing: border-box; A value for box-sizing that includes padding and border in the element's total width and height. Useful for easier sizing calculations. box-sizing: border-box; `box-sizing`
3. CSS Margin Collapse When vertical margins of adjacent block-level elements collapse, resulting in a margin equal to the larger of the two adjacent margins. .block-one { margin: 5px; } .block-two { margin: 15px; } Margin properties
4. CSS auto keyword A value used in various properties (e.g., margin, width) to automatically calculate a property's value based on the context or content. margin: 0 auto; Various
5. Dealing with overflow How CSS handles content that overflows the box dimensions. Options include visible (default), hidden, scroll, and auto. overflow: hidden; overflow
6. Height and Width Maximums/Minimums CSS properties (max-height, min-height, max-width, min-width) that set limits on the size of an element, ensuring it doesn't exceed or fall below specified dimensions. max-width: 500px; max-height, min-height, max-width, min-width
7. The visibility Property Determines whether an element is visible (visible) or hidden (hidden). Unlike display: none;, the element still takes up space in the layout when visibility: hidden;. visibility: hidden; visibility

Display and Positioning

Topic Description Example Applicable Properties
1. CSS z-index property Specifies the stack order of an element. Elements with higher z-index values are displayed in front of elements with lower values. z-index: 1; z-index
2. Fixed CSS Positioning Positions an element relative to the browser window, so it stays fixed even when scrolling. position: fixed; position
3. CSS display property Specifies how an element should be displayed. Common values include block, inline, flex, and more. display: flex; display
4. CSS position: absolute; Positions an element relative to its closest positioned ancestor instead of positioned relative to the viewport, like fixed.. position: absolute; position
5. CSS position: relative; Makes an element positioned relative to its normal position in the document flow, without changing the layout. position: relative; position
6. CSS float property Specifies whether an element should be placed to the left or right of its container, allowing text and inline elements to wrap around it. float: left; float
7. The CSS clear property Specifies which sides of an element can have floating elements. It prevents elements from wrapping around a floated element. clear: both; clear

Colors

2. CSS Hexadecimal Colors Specifies colors using a hexadecimal notation. The format includes three pairs of hex digits representing the intensities of red, green, and blue components. #00ff00; color
3. CSS HSL Colors Describes colors using the Hue-Saturation-Lightness model. Allows more intuitive color selection by specifying the hue, saturation, and lightness. hsl(120, 100%, 50%); color
4. CSS rgb() Colors Specifies colors using the Red-Green-Blue color model. The values range from 0 to 255 for each color component. rgb(255, 0, 0); color
5. Color Name Keywords Uses predefined color names such as "red," "blue," etc. Limited set of named colors available in CSS. color: red; color
